# 결정 기록 (DECISIONS)

기획서의 `[미정]` 항목에 대해, **방향을 확정하지 않고** 슬라이스를 굴리기 위해 쓴
임시값과 그 이유를 기록합니다. 큰 방향 결정은 별도로 물어봅니다.

| 닭 달걀 쿨다운 | **5초** | 시간 시스템 도입 시 "하루 1개" 등으로 바뀔 수 있음. |

## ✅ 확정된 큰 방향 (2026-06, 사용자 승인 "추천대로")

| 결정 | 내용 |
| --- | --- |
| **시간 흐름** | **없음**. 하루/계절 없이 자유 수집. 진엔딩은 "5년 자동"이 아니라 **목표 달성으로 해금**. |
| **보석 ↔ 속성의 돌** | **분리.** 전설의 보석 = 엔딩용 수집품 / 속성의 돌 = 도구·장비 강화 재료. |
| **하이퍼 시리우스메가 보석** | 7 전설의 보석과 **별개**. 제작 끝판왕(최강 장비/장식). |
| **엔딩 3종** | 광산 엔딩(1000층 도달) / 무지개 엔딩(7보석→레인보우→미래 장치) / 진엔딩(무지개 엔딩 후 해금, 원래 집으로). |
| **연구소** | **마을 안 건물.** 예지의 보석·미래 장치 위치. |
| **속성의 돌** | 각 지역/광산 깊은 층에서 드물게 획득 → 도구·장비 강화에 사용(추후 슬라이스). |
