# 결정 기록 (DECISIONS)

기획서의 `[미정]` 항목에 대해, **방향을 확정하지 않고** 슬라이스를 굴리기 위해 쓴
임시값과 그 이유를 기록합니다. 큰 방향 결정은 별도로 물어봅니다.

## 슬라이스 1 — 마을 + 닭 달걀 + 상점 판매

| 항목 | 임시 결정 | 비고 / 나중에 정할 것 |
| --- | --- | --- |
| 시간·계절 흐름 (4장) | **없음** (자유 모으기) | 아직 미구현일 뿐, "시간 있는 게임"으로 갈지는 미확정. |
| 화폐 단위 (3장) | **골드(G)** | 이름·환율 등 상점 구조는 추후. |
| 달걀 판매가 | **50G** | 가공·경제 밸런스는 추후 일괄 조정. |
| 닭 달걀 쿨다운 | **5초** | 시간 시스템 도입 시 "하루 1개" 등으로 바뀔 수 있음. |
| 모바일 터치 | **슬라이스 1 제외** (키보드/게임패드만) | 기획서의 "Godot 4.7 내장 VirtualJoystick"은 **엔진 기본 노드가 아님** → 슬라이스 2~3에서 직접 구현 예정. |
| 플레이어 비주얼 | `_draw()` 색 도형 (임시) | 카이로소프트풍 픽셀 스프라이트로 교체 예정. |
| 한글 폰트 | **Galmuri11** (OFL, 픽셀 한글) | `assets/fonts/`. 도트 느낌과 한국어 UI를 동시에 충족. |
| 카메라 | 없음 (월드=화면 640×360) | 마을이 커지면 `Camera2D` 추가. |

## 슬라이스 2 — 소·치즈 가공 + 상점 UI + 모바일 터치

| 항목 | 임시 결정 | 비고 |
| --- | --- | --- |
| 우유 판매가 | **80G** | 밸런스 추후 일괄 조정. |
| 치즈 판매가 | **180G** | 가공품이 원재료보다 비싸게(우유 80→치즈 180). |
| 소 우유 쿨다운 | **8초** | 닭(5초)보다 김. |
| 가공대 가공시간 | **6초** | 우유 1 → 치즈 1. |
| 가공대 동작 | 한 번에 1개씩, 즉시 투입·시간 후 자동 산출 | 큐/대량가공은 추후. |
| 상점 판매 | 품목별 "전부 판매" 버튼(보유·단가 표시) | 수량 선택 판매는 추후. |
| 모바일 터치 | 왼쪽 가상 조이스틱(동적 위치) + 오른쪽 "행동" 버튼 | 항상 표시. `emulate_touch_from_mouse=on`이라 데스크톱 마우스로도 동작. |
| 터치 조이스틱 = 엔진 내장? | **아니오, 직접 구현** | 기획서의 "Godot 4.7 내장 VirtualJoystick"은 실제 내장 노드가 아님. |

## 슬라이스 3 — 밭농사(작물)

| 항목 | 임시 결정 | 비고 |
| --- | --- | --- |
| 첫 작물 | **토마토** | 나중에 피클(채소) 가공 재료로 연결 가능. |
| 씨앗 값 | **10G** (심을 때 차감) | 돈→씨앗→작물→돈 루프. 돈 없으면 못 심음. |
| 토마토 판매가 | **40G** | 원물은 저렴, 가공하면 가치 ↑ 방향. |
| 성장 시간 | **12초(실시간)** | ⚠️ 하루/계절 시스템을 아직 안 정해서 임시로 초 단위. 시간 시스템 도입 시 "N일"로 교체. |
| 밭 칸 수 | 3칸 | 추후 목수/확장으로 늘릴 수 있음. |
| 씨앗 구매처 | 없음(심을 때 바로 돈 차감) | 상점에 "구매" 탭은 추후. |

## 슬라이스 4 — 새 지역(숲) + 지역 이동

| 항목 | 임시 결정 | 비고 |
| --- | --- | --- |
| 지역 이동 방식 | **표지판(RegionGate) 상호작용 → 씬 전환** (`change_scene_to_file`) | 인벤토리·돈은 autoload라 유지됨. |
| 첫 새 지역 | **숲** (산딸기 채집) | 기획서의 숲의 보석 경로로 이어질 자리. |
| 산딸기 | 채집물, 판매 **30G**, 쿨다운 6초 | 나중에 잼 재료로 연결. |
| 각 지역 UI | HUD·터치 컨트롤을 지역 씬마다 인스턴스 | 상점/상점메뉴는 마을 전용. 추후 영속 UI로 리팩터 가능. |
| 지역 접근 조건 | 없음(자유 이동) | 광활한 바다=배 필요 등은 추후. |
| 게이트 위치(마을) | 우상단 "숲으로" | 터치 "행동" 버튼과 겹치지 않게 배치. |

## 슬라이스 5 — 광산(채굴) + 광물

| 항목 | 임시 결정 | 비고 |
| --- | --- | --- |
| 광산 방식 | 광맥 바위 상호작용 → 광물 1개(전투 없음) | 바위는 채굴 시 사라짐, 6개 스폰. |
| 내려가기 | "더 내려가기" 사다리 → 깊이 +1, 바위 재생성(같은 씬) | 1000층은 추후. 씬 리로드 없이 깊이만 증가. |
| 깊이 유지 | `GameState.mine_depth` (지역 오가도 유지) | |
| 광물 종류·가격 | 돌5 · 석탄15 · 구리광석25 · 철광석50 · 금광석120 | **석탄 추가**(기획서 9장에서 빠져 있던 것 보완). |
| 깊이별 산출 | 1~2층 돌/석탄/구리, 3층+ 철, 5층+ 금 | 마/오팔/시리우스/이리듐은 더 깊은 층에 추후. |
| 곡괭이 등급(깊이 제한) | **없음(임시)** | 기획서의 도구 강화 게이팅은 추후. |
| HUD/상점 표시 | **보유한 아이템만 표시**로 변경 | 광물 등 종류가 늘어 확장성 확보. 상점은 보유한 것만 판매. |

## 아직 사람에게 물어볼 큰 결정 (기획서 15장)

- 하루·계절 흐름 여부 (게임 구조에 큰 영향)
- 밭농사·숲 채집 도입 → 잼·피클·꿀 재료 연결
- 속성의 돌 용도 / 7 전설 보석과의 관계
- "하이퍼 시리우스메가 보석" ↔ 전설의 보석 관계
- 토캣몬 배경 / 진엔딩 조건("5년" 자동인지 해금인지)
- 연구소 위치(마을 안 vs 별도 지역)
