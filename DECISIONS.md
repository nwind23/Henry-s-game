# 결정 기록 (DECISIONS)

기획서의 `[미정]` 항목에 대해, **방향을 확정하지 않고** 슬라이스를 굴리기 위해 쓴
임시값과 그 이유를 기록합니다. 큰 방향 결정은 별도로 물어봅니다.

## 슬라이스 1 — 마을 + 닭 달걀 + 상점 판매

| 항목 | 임시 결정 | 비고 / 나중에 정할 것 |
| --- | --- | --- |
| 시간·계절 흐름 (4장) | **없음** (자유 모으기) | 아직 미구현일 뿐, "시간 있는 게임"으로 갈지는 미확정. |
| 화폐 단위 (3장) | **골드(G)** | 이름·환율 등 상점 구조는 추후. |
| 달걀 판매가 | **50G** | 가공·경제 밸런스는 추후 일괄 조정. |
| 닭 달걀 쿨다운 | **5초** | 시간 시스템 도입 시 "하루 1개" 등으로 바뀔 수 있음. |
| 모바일 터치 | **슬라이스 1 제외** (키보드/게임패드만) | 기획서의 "Godot 4.7 내장 VirtualJoystick"은 **엔진 기본 노드가 아님** → 슬라이스 2~3에서 직접 구현 예정. |
| 플레이어 비주얼 | `_draw()` 색 도형 (임시) | 카이로소프트풍 픽셀 스프라이트로 교체 예정. |
| 한글 폰트 | **Galmuri11** (OFL, 픽셀 한글) | `assets/fonts/`. 도트 느낌과 한국어 UI를 동시에 충족. |
| 카메라 | 없음 (월드=화면 640×360) | 마을이 커지면 `Camera2D` 추가. |

## 아직 사람에게 물어볼 큰 결정 (기획서 15장)

- 하루·계절 흐름 여부 (게임 구조에 큰 영향)
- 밭농사·숲 채집 도입 → 잼·피클·꿀 재료 연결
- 속성의 돌 용도 / 7 전설 보석과의 관계
- "하이퍼 시리우스메가 보석" ↔ 전설의 보석 관계
- 토캣몬 배경 / 진엔딩 조건("5년" 자동인지 해금인지)
- 연구소 위치(마을 안 vs 별도 지역)
